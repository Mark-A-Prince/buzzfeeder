About

Hello, welcome to BuzzFeeder! This is a website dedicated to keeping a top ten list of numbers used by the website Buzzfeed. Basically, my brother and I wanted to figure out what numbers Buzzfeed used for their listicles and see if there was a pattern/strategy for it. The results below are from a python script I made which crawls through most of the sections on Buzzfeed, extracts the titles, and then uses a bash script to count the instances of all numbers from 1 - 100. Anyways, hope you enjoy it. If you want to know more about the script/try it yourself, check out the github repository here.
